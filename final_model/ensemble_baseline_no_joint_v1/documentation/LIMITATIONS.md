# 制約と注意点

- mixed_finalの結果を確認したうえで最終モデルを選択しているため、mixed_finalは完全な未知テストではない。
- 低重複pseudo_final 7セットでは6セットを完全分離したが、pseudo_final_01はAUC 0.7917、分離マージン-0.0532だった。
- pseudo_final_03は順位では完全分離したが、実データ1件が0.5未満だった。
- 確率0.5は学習目的関数で直接最適化された閾値ではなく、確率の較正が全時代で一定とは限らない。
- jointあり・なしの両方が同じ方向に誤るデータは、単純平均では修正できない。
- モデルはSP500とDGS10、1260日窓を前提としている。
