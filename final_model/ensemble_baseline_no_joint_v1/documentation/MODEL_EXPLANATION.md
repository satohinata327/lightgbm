# モデル説明

## 構成

2つのLightGBM二値分類器が、それぞれ実データである確率を出力する。

1. jointありモデル：26特徴量
2. jointなしモデル：25特徴量

学習条件は`window_length=1260`、`stride=20`、生成データはZAとMF2を各100系列、実データは時系列順のblocked CVである。B5は学習に使用せず最終テストにのみ使用した。

最終確率は2モデルの確率平均である。jointありモデルは同時極端変動を利用して一部の生成データを強く排除し、jointなしモデルはjointの数日差に対する感度を緩和する。異なる誤り方を平均することで、片方だけが高く評価した候補を抑制する。
