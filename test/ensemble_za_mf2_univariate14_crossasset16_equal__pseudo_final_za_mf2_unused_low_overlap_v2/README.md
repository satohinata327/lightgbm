# 0.5–0.5アンサンブルの擬似final評価

単系列14特徴量モデルと複数資産間16特徴量モデルを等重みでsoft votingした結果である。

## 結果

- 7セットすべてでAUC 1.0
- 7セットすべてで実データ2件が確率上位2件
- 7セットすべてで「最低の実データ確率 > 最高の生成データ確率」
- 生成データが0.5を超えたセットは0件
- 実データ2件が両方0.5を超えたセットは4件

したがって、順位付けと実データ・生成データ間の分離は全セットで成功した。一方、`pseudo_final_05`〜`07`では実データの一方が0.4882であり、固定閾値0.5では取りこぼす。これは判別順位の失敗ではなく、確率尺度・閾値の問題である。

全体結果は `suite_summary.json`、セット別結果は各 `pseudo_final_XX/` に保存している。
