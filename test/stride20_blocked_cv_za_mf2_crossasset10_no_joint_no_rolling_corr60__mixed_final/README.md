現在の11特徴量モデルから`joint_large_move_95`だけを削除してmixed_finalを解析した。

| 順位 | mask | 実データ確率 | 0.5判定 |
|---:|---|---:|---|
| 1 | mask3 | 0.6655 | 実データ |
| 2 | mask6 | 0.6643 | 実データ |
| 3 | mask4 | 0.6637 | 実データ |
| 4 | mask2 | 0.5373 | 実データ |
| 5 | mask10 | 0.4291 | 生成データ |
| 7 | mask7 | 0.3874 | 生成データ |
| 14 | mask1 | 0.3207 | 生成データ |

jointなしでもmask10は0.5未満になった。一方、mask7が0.3874へ低下し、mask2が0.5373へ上昇したため、狙っている4maskの完全分離には失敗した。現在の特徴量集合では、jointはmask10を排除することよりも、mask7を実データ側へ保ちmask2を生成データ側へ抑える役割が大きい。
