# 等重みsoft voting：低重複擬似final v2評価

B5実データ窓の最大重複率を89.76%に抑え、baseline学習・旧pseudoのどちらにも未使用のZA・MF2系列を使用した7セットの結果である。アンサンブル重みはbaseline 0.5、jointなし0.5から変更していない。

|---|---:|

| 実データ2件がTop-2 | 6/7 |
| 実データ2件がTop-4 | 6/7 |

Set 02〜07はAUC 1.0で、実データ2件を生成データ12件より完全に上位へ置いた。Set 03は一方の実データ確率が0.4770で0.5未満、Set 05は0.5002だったが、順位上は完全分離できている。

Set 01だけはAUC 0.7917、Top-2実データ1件、マージン-0.0532だった。実データは2013-11-15〜2018-12-13が0.413、2020-09-01〜2025-10-02が0.228であり、以前確認したB5両端付近の弱点が再現した。

旧10セットより実データ窓の重複を抑えた評価では、等重みアンサンブルは6/7セットで完全分離した。0.5閾値より順位・マージンの方が安定している。
