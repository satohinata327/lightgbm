`sp_std`と`rate_std`を除いた単系列12特徴量モデルを0.3、複数資産間16特徴量モデルを0.7でsoft votingした結果である。

- 平均AUC: 0.9940
- 実データ2件が確率上位2件に入ったセット: 6/7
- 全実データが上位4件に入ったセット: 7/7
- 最低実データ確率が最高生成データ確率を上回ったセット: 6/7
- 実データ2件がともに0.5を超えたセット: 4/7
- 0.5を超える生成データがなかったセット: 6/7
