# 0.5–0.5アンサンブルのmixed_final解析

単系列14特徴量モデルと複数資産間16特徴量モデルの実データ確率を0.5ずつ平均した。

## 主な結果

| 順位 | mask | 実データ確率 | 0.5判定 |
|---:|---|---:|---|
| 1 | mask6 | 0.6832 | 実データ |
| 2 | mask3 | 0.6722 | 実データ |
| 3 | mask7 | 0.6469 | 実データ |
| 4 | mask10 | 0.6093 | 実データ |
| 5 | mask4 | 0.5391 | 実データ |
| 6 | mask1 | 0.4663 | 生成データ |

狙っている `mask3,4,6,7` はすべて0.5を超え、確率上位5件に入った。また、複数資産間モデル単体では0.5885だった `mask1` は、単系列モデルの0.3441と平均することで0.4663まで下がった。一方で `mask10` は両モデルが高く評価しており、0.6093の偽陽性として残る。

詳細な全maskの確率は `predictions.csv`、グラフは `prob_real_ranking.svg`、各構成モデルの結果は `component_univariate14/` と `component_crossasset16/` に保存している。
