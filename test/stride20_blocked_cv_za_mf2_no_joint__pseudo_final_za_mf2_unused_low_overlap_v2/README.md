# no_joint単体モデルの低重複擬似final評価

baseline26特徴量から`joint_large_move_95`だけを除いた25特徴量モデルを、アンサンブルせず単体で評価した結果である。

- 平均AUC: 0.9524
- 実データ2件が確率上位2件: 5/7セット
- 最低実データ確率が最高生成データ確率を上回る: 5/7セット
- 全実データが上位4件以内: 5/7セット
- 実データ2件がともに0.5超え: 5/7セット
- 0.5を超える生成データなし: 7/7セット

`pseudo_final_01`と`03`で順位分離に失敗した。一方、生成データは全セットで0.5未満に抑えられている。
