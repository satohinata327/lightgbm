# smooth joint 11特徴量モデル：mixed_final評価

`joint_large_move_95`を、95%点付近を連続的に重み付けする`joint_large_move_rank95_smooth_tau02`へ置き換えたモデルの評価結果である。

## 結果

| 順位 | mask | 実データ確率 |
|---:|---|---:|
| 1 | mask6 | 0.7332 |
| 2 | mask3 | 0.7281 |
| 3 | mask4 | 0.7177 |
| 4 | mask7 | 0.6067 |
| 5 | mask10 | 0.6040 |
| 6 | mask9 | 0.5462 |

目的としているmask3・4・6・7は上位4件に並んだ。ただし、0.5閾値ではmask10とmask9も実データ判定になる。hard joint版は目的の4件とその他を0.5で分離できていたため、mixed_finalへの適合だけを見るとsmooth joint版は悪化している。

これは、95%点を超えた同時変動日数の小さな違いを滑らかに扱う代わりに、mask10を強く押し下げていた離散的な分岐も弱くなったためと解釈できる。
