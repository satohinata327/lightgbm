# ZA＋MF2：B3・B4実データ重み付け実験

ZA＋MF2 baselineに対し、学習時の実データだけを次の相対重みへ変更した。

- B1：1.0
- B2：1.0
- B3：1.2
- B4：1.5

正例クラス全体を重くする効果を避けるため、各foldの学習データ内で実データ重みの平均が1になるよう正規化した。生成データの重みは1で固定した。ZA・MF2の系列、26特徴量、block、LightGBMパラメータはbaselineと同じである。

最終学習時の実効重みはB1 0.8511、B2 0.8511、B3 1.0213、B4 1.2766である。

AUCとloglossはほぼ同等だが、0.5閾値でのB5 RecallとBalanced accuracyは悪化した。
