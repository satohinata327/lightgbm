# ZA＋MF2：joint 90〜99%等重み平均

現行ZA＋MF2 baselineを対照とし、`joint_large_move_95`だけを90〜99%の10段階を等重みで平均した`joint_large_move_90_99_mean`へ置換した。特徴量数は26のままで、実データblock、ZA・MF2の系列割当、`window_length=1260`、`stride=20`、LightGBMパラメータ、early stopping、seedは同一である。Optunaは使用していない。

## 定義

各qについて、SP500とDGS10の絶対値がそれぞれのq分位点を同時に超えた割合を`joint_q`とする。

```text
joint_large_move_90_99_mean
  = mean(joint_90, joint_91, ..., joint_99)
```

100%分位点を厳密に超える観測は存在しないため、100%は含めていない。各分位点の値は学習入力にはせず、診断CSVだけに保存している。

## Joint診断

| mask | joint 95% | 90〜99%平均 |
|---|---:|---:|
| mask1 | 0.007143 | 0.009444 |
| mask4 | 0.007937 | 0.010238 |
| mask9 | 0.006349 | 0.009603 |
| mask10 | 0.006349 | 0.010000 |

90〜99%全体ではmask1・4・9・10が非常に近い。95%の単一点にあった8日対9日の不連続は弱まったが、この平均値だけではこれらを分離できないことが分かった。

新jointのgain比率は26.0%で、baselineの旧jointの26.4%とほぼ同じだった。一方、同じ閾値の反復回数はbaselineの最大26回から新モデルでは最大9回へ減少し、単一閾値への集中は緩和された。

## 判断

複数分位点平均によって、単一の8日・9日境界への依存を分散させるという技術的目的は達成した。しかしCV平均とmixed finalの分離は悪化したため、この形ではbaselineを置き換えない。次にjointを改良する場合は、90〜99%平均をさらに調整するよりも、同時変動の正負方向または超過強度を区別する方が有力である。
