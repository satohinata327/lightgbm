# ZA + MF2 複数資産間11特徴量モデル

直前の15特徴量モデルから、mask13を強く実データ側へ押し上げていた60日ローリング相関の要約4特徴量を削除した対照実験である。

- `rolling_corr60_std`
- `rolling_corr60_q10`
- `rolling_corr60_q50`
- `rolling_corr60_q90`

すでに削除済みの`corr_abs_sp_rate`と`rate_lead_sp_corr_1`も使用しない。`corr_rolling_vol20_sp_rate`は20日ローリング・ボラティリティ同士の相関であり、60日リターン相関とは異なるため残している。

学習データ、生成データ割り当て、blocked CV、LightGBMパラメータは直前の15特徴量モデルと同一である。

## 学習結果

| 指標 | 15特徴量版 | 60日ローリング相関削除後 |
|---|---:|---:|
| CV平均AUC | 0.6642 | 0.7710 |
| CV最小AUC | 0.3967 | 0.6331 |
| B5 AUC | 0.9764 | 0.9836 |
| B5 logloss | 0.4303 | 0.4049 |
| B5実データRecall | 0.9294 | 0.9176 |
| B5生成データSpecificity | 0.9550 | 0.9450 |

60日ローリング相関削除後は、CVとB5の順位性能、loglossが15特徴量版より回復した。実データRecallと生成データSpecificityはわずかに低下した。
