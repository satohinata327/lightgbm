# ZA＋MF2：joint削除対照実験

ZA＋MF2 baselineの26特徴量から`joint_large_move_95`だけを削除した25特徴量モデルである。生成系列、実データblock、seed、LightGBMパラメータはbaselineと同じである。

## Blocked CVとB5

| 指標 | jointなし | baseline |
|---|---:|---:|
| CV平均AUC | 0.7361 | 0.7657 |
| CV AUC標準偏差 | 0.1646 | 0.2114 |
| CV最低AUC | 0.5512 | 0.5426 |
| 木の本数 | 22 | 36 |
| B5 AUC | 0.9531 | 0.9654 |
| B5 Balanced accuracy | 0.8800 | 0.9060 |
| B5実データRecall | 0.8000 | 0.8471 |
| B5生成データSpecificity | 0.9600 | 0.9650 |
| B5 Logloss | 0.4520 | 0.3354 |

通常のCV・B5指標はbaselineよりやや低下した。

| 指標 | jointなし | baseline |

`joint_large_move_95`を削除すると、低確率だった実データ窓の最低確率は約0.19〜0.21から約0.30〜0.33へ上昇した。Set 01のマージンは-0.2891から-0.0240へ大幅に改善した。8日と10日付近の不連続な判定は緩和された。

一方、完全分離セット数とTop-2成功数は増えておらず、問題を完全には解決していない。

## Mixed final

mask6（0.7382）、mask3（0.7252）、mask7（0.5638）、mask10（0.5435）、mask4（0.5255）が0.5を超えた。目的のmask4よりmask10が上位となり、baselineが持っていたmask3・4・6・7だけの分離は失われた。

joint削除は擬似finalの安定性を改善したが、baselineを全面的に置き換える結果ではない。baselineとのアンサンブル候補として扱う。
