# Baseline＋jointなし：等重みsoft voting結果

2モデルの実データ確率を0.5ずつ平均した。重みは評価結果から最適化せず、事前に等重みで固定した。

## 擬似final 10セット

| 指標 | Ensemble | Baseline | jointなし |
|---|---:|---:|---:|
| 平均AUC | 0.9167 | 0.9167 | 0.9417 |
| 実データ2件がTop-2 | 4/10 | 4/10 | 4/10 |
| 実データ2件がTop-4 | 7/10 | 6/10 | 8/10 |
| 正の分離マージン | 4/10 | 4/10 | 4/10 |
| 平均分離マージン | -0.0358 | -0.1409 | -0.0285 |
| 実データ2件が両方0.5以上 | 4/10 | 4/10 | 4/10 |
| 生成データがすべて0.5未満 | 10/10 | 5/10 | 8/10 |

Top-2と完全分離数は増えなかったが、baselineより平均マージンが大きく改善し、全10セットで生成データ12件をすべて0.5未満に抑えた。jointなし単体より平均AUCとTop-4は低い。

0.5閾値の分類性能とAUCはbaselineをわずかに上回ったが、loglossはbaselineより悪い。

この10擬似finalはすでに開発中に確認しているため、等重み以外の重み最適化には使用しない。採用判断には残りの未使用pathによる新しい評価セットが必要である。
